# Developer Portfolio

A simple developer portfolio built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting started

Prerequisites: Node.js and npm installed (LTS recommended).

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview the production build locally
npm run preview
```

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui

## Project structure

- `src/components` — UI components and sections
- `src/pages` — routing pages (home)
- `public` — static assets (favicon, placeholder image)

## Customization

- Update site text in components under `src/components` (Header, Hero, About, Projects, Skills, Contact, Footer).
- Edit theme colors in `src/index.css` (CSS variables) and `tailwind.config.ts`.
- Replace `public/placeholder.svg` with your own Open Graph image and update meta tags in `index.html`.
